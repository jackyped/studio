# **App Name**: MediChain

## Core Features:

- Admin Dashboard: User-friendly dashboard for admins to manage users, pharmacies, drivers, products, orders, finance and system settings.
- Customer App: Intuitive mobile interface for customers to browse pharmacies, search for medicines, upload prescriptions, track orders and manage their profile.
- Pharmacy App: Mobile app for pharmacies to manage orders, products, documents, store settings and driver assignments.
- Driver App: Simple app for drivers to receive tasks, control task flow with maps supports and wallet management.
- Smart Suggestions: Leverage LLM tool to interpret user reviews and suggest improvements to service.
- Secure User Management: Full support of user registration and login/logout features using secure password management.
- Full configurable settings: Comprehensive system setting for global and environment configuration and customizable system notifications

## Style Guidelines:

- Primary color: Deep sky blue (#3498db) for trust and reliability.
- Background color: Light gray (#ecf0f1) for a clean, modern look.
- Accent color: Emerald green (#2ecc71) to highlight key actions and promotions.
- Body and headline font: 'PT Sans' for readability and a modern feel.
- Use a set of consistent and clear icons throughout all modules, emphasizing clarity.
- Maintain a consistent grid layout across all platforms to ensure intuitive navigation.
- Employ subtle transitions for key interactions.
